# Chess

This project was developed using Angular

## Development server

To start a local development server, run:

```bash
npm run start
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`.

-> Check if Node.js is installed
```bash
node -v
```
If not installed, install with below command:
```bash
npm install
```

-> check Angular version
```bash
ng version
```
-> If not installed, Install Angular CLI globally
```bash
npm install -g @angular/cli
```

-> create new angular project called "chess"
```bash 
ng new chess
```

-> create following:
"chess-logic" folder in src/app/chess-logic
"pieces" folder in src/app/chess-logic/pieces

src/app/chess-logic/pieces/knight.ts
src/app/chess-logic/pieces/pawn.ts
src/app/chess-logic/pieces/piece.ts
src/app/chess-logic/pieces/queen.ts
src/app/chess-logic/pieces/rook.ts
